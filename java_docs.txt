9) ArrayList
Difference between arraylist and array is that in arraylist size of array can be modified
while in array size is fixed.

-> Creating an ArrayList

-> Adding an element
arr.add("aman");

-> Accessing an item
arr.get(index);
arr.get(0); #aman

-> Modifying an element

arr.set(0, "ayushi");

-> Removing an element

-> Size

-> Sorting
